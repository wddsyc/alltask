# Merge Sorted Array
`简单`

## JavaScript
```javascript
function merge(nums1, m, nums2, n) {
  while (m > 0 && n > 0) {
    if (nums1[m - 1] < nums2[n - 1]) {
      nums1[m + n - 1] = nums2[n - 1]
      n -= 1
    } else {
      nums1[m + n - 1] = nums1[m - 1]
      m -= 1
    }
  }
  while (n-- > 0) nums1[n] = nums2[n]
}
```
